Arif Chat Widget Demo

1. UMD Version (Global)

Include the widget via script tag:

<script src="./arif-chat-widget.umd.js"></script> <link rel="stylesheet" href="./arif-chat-widget.css">

2. IIFE Version (Immediate)

Self-executing version:

<script src="./arif-chat-widget.min.js"></script> <link rel="stylesheet" href="./arif-chat-widget.css">

3. ES Module Version

For modern bundlers:

import { ChatbotWidget } from './arif-chat-widget.es.js'; const widget = new ChatbotWidget(); widget.init({ tenantId: 'demo-tenant', apiUrl: 'http://localhost:8000', websocketUrl: 'ws://localhost:8000/chatHub' });

4. Auto-initialization via Script Tag

<script src="./arif-chat-widget.umd.js" data-chatbot-config='{"tenantId":"demo-tenant","apiUrl":"http://localhost:8000"}'> </script>
